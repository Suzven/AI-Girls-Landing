============================================================
CompanionVerse — установка
============================================================

СОСТАВ:
  index.php            — главная страница
  tool.php             — карточка оффера (tool.php?slug=candy-ai)
  category.php         — страница категории (category.php?slug=ai-girlfriend)
                         и фильтр Free Trial (category.php?free=1)
  search.php           — живой поиск (AJAX)
  config.php           — КРЕДЫ ОТ БД ВПИСЫВАТЬ СЮДА
  schema.sql           — структура БД + демо-данные
  management-panel/    — панель управления (/management-panel/)
  uploads/             — сюда падают загруженные картинки
  assets/              — стили и скрипты

------------------------------------------------------------
ШАГ 1. База данных
------------------------------------------------------------
1. В панели хостинга создай базу данных MySQL.
   Рекомендуемое имя: companionverse (можно любое).
2. Открой phpMyAdmin → выбери базу → вкладка «Импорт» →
   загрузи файл schema.sql → «Вперёд».
   Создадутся все таблицы, категории и демо-оффер Candy AI.

Таблицы, которые создаются:
  users           — логины для панели (username, password)
  tools           — офферы
  categories      — категории
  tool_categories — связь оффер–категория
  badges          — быстрые бейджи
  features        — key features
  pricing_plans   — тарифы (название/цена)
  pros / cons     — плюсы и минусы
  best_for        — блок Best For
  faqs            — вопросы-ответы
  alternatives    — связь оффер–альтернатива

------------------------------------------------------------
ШАГ 2. Креды
------------------------------------------------------------
Открой config.php и впиши свои данные:

  define('DB_HOST', 'localhost');
  define('DB_NAME', 'companionverse');   // имя твоей БД
  define('DB_USER', 'юзер_бд');
  define('DB_PASS', 'пароль_бд');

------------------------------------------------------------
ШАГ 3. Заливка на хостинг
------------------------------------------------------------
Залей все файлы в корень сайта (public_html или www).
Папке uploads/ дай права на запись (обычно 755 или 775),
иначе не будет работать загрузка картинок из панели.

------------------------------------------------------------
ШАГ 4. Вход в панель
------------------------------------------------------------
Адрес:  https://твой-домен/management-panel/

Демо-доступ (создаётся schema.sql):
  логин:  admin
  пароль: admin123

СМЕНИ ПАРОЛЬ: в phpMyAdmin открой таблицу users и поменяй
поле password. Можно вписать пароль простым текстом —
сайт понимает и простой текст, и password_hash().
Новых юзеров добавляй туда же: просто новая строка
(username + password).

------------------------------------------------------------
Как добавлять офферы
------------------------------------------------------------
Панель → «+ Добавить оффер». Все блоки карточки заполняются
из одной формы:
  — основное (название, tagline, overview, ссылки)
  — логотип и один скриншот (URL или загрузка файла)
  — editor score и метки (Featured / Verified / New / Popular / Free Trial)
  — категории (галочки)
  — бейджи, features, плюсы, минусы, best for
    (одна строка = один пункт)
  — тарифы: пары «название / цена»
  — FAQ: пары «вопрос / ответ»
  — альтернативы: галочками из уже добавленных офферов

Оффер автоматически появляется на главной (Latest),
в Featured (если поставлена галочка), во всех своих
категориях, в поиске и в блоках альтернатив.
